Introduzione a Bootstrap

Bootstrap è un framework front-end che aiuta a creare siti web responsivi e ben strutturati.

1. Cos'è un CDN?

Un CDN (Content Delivery Network) è una rete di server distribuiti che permette di servire file come CSS e JavaScript più rapidamente e con maggiore affidabilità.

2. Vantaggi del CDN rispetto a scaricare Bootstrap sul proprio sito:

  • Velocità: I file vengono scaricati dal server più vicino all'utente, migliorando i tempi di caricamento.
  • Affidabilità: Se un server del CDN è offline, gli utenti possono comunque ottenere i file da un altro server.
  • Caching: Il browser dell'utente può memorizzare i file localmente, migliorando i tempi di caricamento nelle visite future.

3. A cosa serve il tag meta con "viewport"?

L'attributo content="width=device-width, initial-scale=1" serve a garantire che la pagina si adatti correttamente ai dispositivi mobili, evitando lo zoom automatico e ottimizzando la visualizzazione sui vari schermi.

4. Differenza tra Container e Container Fluid:

Container: È un contenitore a larghezza fissa che si adatta a diverse risoluzioni di schermo, ma ha una larghezza massima per evitare che il contenuto diventi troppo largo su schermi grandi.

Container Fluid: È un contenitore che occupa sempre il 100% della larghezza disponibile, senza limiti massimi di larghezza.
